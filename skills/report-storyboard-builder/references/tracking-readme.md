# Report Storyboard Tracking Templates

## Purpose

These templates keep the report structure explicit enough that drafting can start without guesswork.

## Template roles

- `report-storyboard-template.md`: use for section purpose, claim, proof obligation, evidence, implication, exclusions, and dependencies
- `section-file-plan-template.md`: use for ordered section filenames and draft dependencies
- `open-questions-template.md`: use for unsupported sections and unresolved proof needs

## Working rule

Keep the live versions in `tracking/`. Update them while sequencing the report.
