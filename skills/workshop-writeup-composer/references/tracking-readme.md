# Workshop Write-up Tracking Templates

## Purpose

These templates keep the workshop reconstruction inspectable instead of relying on memory or chat narrative.

## Template roles

- `workshop-writeup-template.md`: use for session framing, discussion themes, decisions, actions, and unresolved issues
- `workshop-evidence-map-template.md`: use for claim-to-evidence traceability
- `open-questions-template.md`: use for missing transcript detail, unresolved issues, and follow-up checks
- `decision-log-template.md`: use for confirmed decisions and their evidence basis

## Working rule

Keep the live files under `tracking/`. Update them while reconstructing the workshop, not after publication.
