# Findings Workbook

Use this workbook when a document has many discrete findings and the final report should not be the only place those findings exist.

## Usage

- Create one row per finding candidate.
- Keep this more explicit than the report body.
- Update IDs only when the structure is stable enough to stop renumbering everything.
- Record confidence where the evidence is indirect or incomplete.
- Record where each finding landed in the final report.

| ID | Area | Plain statement | Impact | Consequence if unchanged | Evidence refs | Confidence | Rating | Recommended action | Report section |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| F-01 | Example | State the finding plainly | What changes in practice | What happens if nobody acts | file or source refs | High / Medium / Low | Critical / High / Medium / Low | Short action statement | Section name or appendix ID |
