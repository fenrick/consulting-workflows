# Presentation Storyboard Tracking Templates

- `presentation-storyboard-template.md`: use for slide number, title, takeaway, evidence payload, presenter note, transition, and risk note
- `slide-map-template.md`: use for opening, analysis, and closing sequence planning
- `open-questions-template.md`: use for unsupported slides and unresolved proof needs

Keep working files under `tracking/`. Update them during sequencing, not after the deck is drafted.
