# Presentation Storyboard Term Sheet

- `slide takeaway`: the one sentence the audience should retain from a slide
- `evidence payload`: the proof element a slide depends on
- `transition`: the link from the prior slide to the current one
- `opening sequence`: context and objective-setting slides
- `analysis sequence`: the core proof and implication slides
- `closing sequence`: decision, action, or next-step slides

## Usage rules

- Prefer `slide takeaway` over `message` when referring to the audience-facing point.
- Prefer `evidence payload` over `content` when discussing proof.
- Keep `transition` explicit in notes, not implied.
