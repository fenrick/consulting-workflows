# Input Evidence Cataloger Workflow

1. Inventory all files under `inputs/`.
2. Build `inputs/processed/document-map.md` for folder/archive structure.
3. Extract factual statements into `tracking/review-notebook.md`.
4. Register provenance and supersession in `tracking/source-register.md`.
5. Record contradictions and missing metadata in `tracking/open-questions.md`.
6. Re-run a contradiction pass before handoff.
