# Input Evidence Catalog Term Sheet

## Purpose

Use this sheet to keep intake language stable while the corpus is being mapped and classified.

## Core terms

- `source register`: the master inventory of inputs and their metadata
- `review notebook`: extracted factual observations and confidence notes
- `document map`: the folder and archive structure plus supersession relationships
- `provenance`: what can be established about origin, author, date, or custody
- `supersession`: one file replacing or updating another
- `contextual source`: a source used for framing, not direct proof

## Usage rules

- Prefer `source register` over `inventory sheet`.
- Prefer `review notebook` over `notes` when referring to extracted facts.
- Use `contextual source` only where a file is not safe for direct claim support.
- Keep provenance statements separate from content summaries.
