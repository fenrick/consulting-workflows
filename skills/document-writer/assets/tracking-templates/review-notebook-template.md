# Review Notebook

Use this during source intake and evidence review. It is the running memory of the document stream.

This file is not the polished report. It is where observations, insights, emerging findings, and follow-up checks should be captured as they appear so later passes do not have to rediscover them from first sources.

## Usage

- Add notes as sources are processed, not at the end.
- Mark uncertainty instead of waiting for certainty.
- Keep observations tied to a source wherever possible.
- Promote stable items into the findings workbook or report body when they are mature enough.
