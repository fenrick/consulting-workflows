# Evidence Consistency Audit Tracking Templates

## Purpose

These templates keep the audit inspectable instead of burying contradiction handling in ad hoc notes.

## Template roles

- `evidence-consistency-report-template.md`: use for issue IDs, claim references, source locations, severity, confidence, and resolution state
- `open-questions-template.md`: use for unresolved contradictions, proof gaps, and next steps
- `decision-log-template.md`: use for accepted risks and audit closure decisions

## Working rule

Keep the working versions in `tracking/`. Update them during the audit, not after it.
